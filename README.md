## About me

I started my career on the visual side of digital products—thinking about layouts, user journeys and how people interact with software.

Today, I also work on the technical side: writing application logic, connecting databases, handling user actions and turning ideas into working products.

That journey from **designing interfaces** to **building complete applications** shapes the way I approach development.

> I care about two things: whether the software works and whether people can understand how to use it.

I recently completed a **Diploma in Full Stack Software Development with Predictive Analytics** at Code Institute.

## 🏠 Houselytics

### Predictive analytics for property valuation

An interactive application that explores housing data, explains the features influencing sale prices and predicts property values using a trained machine-learning model.

**Technology:**  
`Python` `Streamlit` `Pandas` `Plotly` `scikit-learn`

The project includes exploratory data analysis, hypothesis validation, feature engineering, model evaluation and an interactive valuation tool.

## 🌍 CO₂ Clock

### Climate information made visual

An interactive JavaScript application that presents simulated CO₂ changes, global emissions information and educational climate content through counters, charts and searchable data.

**Technology:**  
`JavaScript` `HTML5` `CSS3` `Chart.js` `Bootstrap`

The application includes an animated CO₂ counter and an emissions dashboard containing information for 70 countries.
